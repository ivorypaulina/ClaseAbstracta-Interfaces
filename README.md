# Ejercicios de Programación Orientada a Objetos en Java

## Descripción
Este repositorio contiene ejercicios desarrollados durante la asignatura de Programación Orientada a Objetos utilizando Java.

Los ejercicios aplican conceptos fundamentales como:
- Clases abstractas
- Herencia
- Interfaces
- Polimorfismo
- Sobrescritura de métodos
- Encapsulamiento
- Arreglos de objetos
- instanceof

---

# Ejercicio 1: Sistema de Vehículos

## Descripción
Implementación de un sistema de vehículos utilizando herencia e interfaces.

## Clases principales
- `Vehiculo` (clase base)
- `Auto`
- `Moto`
- `Camion`
- `Electrico` (interfaz)

## Conceptos aplicados
- Herencia
- Interfaces
- Polimorfismo
- Sobrescritura de métodos
